Acerca de nosotros

Proceso de creación

  • 1.

    Fundición

    se fundirá el metal en un crisol (los cuales pueden ser de grafito o arcilla) para que resistan las elevadas temperaturas que requiere el horno o el soplete durante la fundición. En caso de que no tenga un horno de fundición o solamente se vaya a realizar un anillo, se podrá fundir el metal dentro de un crisol de mango con el soplete directo para mayor practicidad.

  • 2.

    Colado

    Cuando el metal ya obtenga la temperatura adecuada y su estado sea líquido, se verterá en los moldes para darle la figura de una barra. Este proceso recibe el nombre de fundición y colada o colado. Una vez se vierta el metal en el molde, (puede ser de hierro colado, acero, grafito) se debe esperar unos minutos antes de retirarlo para que se solidifique y selle correctamente.

  • 3.

    Laminación

    Como la pieza que se pretende elaborar es un anillo, su estado debe ser plano, por lo que la barra debe ser pasada varias veces por una máquina de prensa o laminación, la cual compactará el metal en forma de hoja con sus rodillos, hasta obtener el grosor deseado para el anillo. Si es necesario se puede utilizar un martillo plástico para dar más forma a la lámina. Se debe dar forma circular a la barra para ir formando el anillo en una máquina especial que va redondeando la lámina.

  • 4.

    Darle Forma

    Una vez soldadas las puntas, ya tendrá la forma circular del anillo y se le debe dar la forma redonda, ya sea en la máquina o con un mandril o barra de anillo, ayudándose con un martillo de plástico o de cuero.

  • 5.

    Lijado o Limado

    Este proceso se hace con las limas especiales que existen en diferentes calibres, para lijar o raspar las imperfecciones que pueda tener el anillo, se debe hacer por lado externo, interno y los bordes. Se pueden utilizar las máquinas de lijado o hacerlo de una forma manual, dependerá de las herramientas con las que se cuente o del gusto del joyero.

  • 6.

    Pulido

    Para finalizar el anillo, es importante el proceso de pulir la pieza ya que esta nos ayuda a darle el acabado brillante e impecable a la joya. Proceso que se realizará con una maquina pulidora o tambor pulidor la cual lleva unos discos que se encarga de pulir el anillo.

  • .7

    Limpieza

    Como último proceso, se deben eliminar los rastros de cera o materiales que quedaron en el anillo. Para esto, utilizaremos una máquina de ultrasonido
